# novel-cover：小说封面生成

根据书名和作者名，分析题材风格，生成含标题和署名的封面图。

## 执行流程

### Step 1：读取项目信息

```bash
export PROJECT_ROOT="$(python -X utf8 "${SCRIPTS_DIR}/novel.py" --project-root "${CLAUDE_PROJECT_DIR:-$PWD}" where)"
```

从 `.novel/state.json` 或直接询问用户获取：
- 书名
- 作者名（笔名）
- 题材（用于判断封面风格）

### Step 2：分析风格方向

根据题材推荐封面风格：
- 都市/重生 → 写实风、城市剪影、冷暖色调对比
- 玄幻/仙侠 → 水墨风、大场景、云雾缭绕
- 科幻/末世 → 科技感、冷色调、机械纹理
- 悬疑/惊悚 → 暗色调、神秘光影、局部特写
- 历史/权谋 → 做旧色调、古风元素、沉稳构图

### Step 3：生成封面

调用 imagegen skill 生成图片，包含：
- 书名（大号醒目字体，居上或居中）
- 作者名（小号字体，居下）
- 符合题材风格背景

如无特殊要求，默认使用系统字体（不嵌入自定义字体包）。

### Step 4：输出

- 输出封面图到项目根目录（`封面.png` 或 `封面.jpg`）
- 展示给用户确认
- 用户不满意可调整参数重新生成

## 参考

本 skill 依赖 imagegen skill 的图像生成能力。
封面风格参考见 story-cover 的 cover-styles.md。
生成封面时 title 和 author 字样直接输出在图片上，不单独生成 PSD 模板。
如果反复生成的封面不符合预期，建议用户提供风格参考图。
封面文件建议保存为 PNG 格式以支持透明背景叠加。
番茄平台封面要求：竖版 600×800 像素，文件名建议 `cover.png`。
